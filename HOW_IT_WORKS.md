## üì± How to Use iChanga

### Step 1: Create Your Account
1. Download or open iChanga
2. Click "New here? Create an account"
3. Enter your email (e.g., user@example.com)
4. Set a secure password (minimum 6 characters)
5. Verify your email through the link sent to your inbox

### Step 2: Create or Join a Group
1. Go to "Groups" section
2. **To Create:** Click "Create New Group"
   - Enter group name (e.g., "Friends Savings Circle")
   - Set a target amount
   - Set contribution deadline
   - Invite members
3. **To Join:** Request to join an existing group

### Step 3: Record Contributions
iChanga offers multiple ways to record contributions:

#### Option A: Manual Entry
1. Go to your group
2. Click "Add Contribution"
3. Enter:
   - Contributor name
   - Amount
   - Phone number
   - Transaction ID (from payment confirmation)
4. Submit for verification

#### Option B: Mobile Money Message
1. Paste the confirmation message from your mobile money service
2. iChanga automatically extracts:
   - Amount paid
   - Transaction ID
   - Timestamp
3. Review details and submit

#### Option C: Screenshot Upload
1. Take a screenshot of your payment confirmation
2. Upload the image
3. Admin reviews and approves
4. Contribution is recorded

### Step 4: Track Progress
- View group dashboard with:
  - Total contributions
  - Members' individual contributions
  - Progress toward group target
  - Contribution charts
- Check your personal achievements and streaks
- View leaderboard rankings

### Step 5: Manage Group (Admins Only)
Admins can:
- Approve or reject contributions
- View audit logs of all actions
- Manage members
- Set contribution reminders
- Generate PDF reports

## üõ°Ô∏è Security & Fraud Prevention

### How We Prevent Scams

1. **Admin Approval System**
   - All contributions require admin verification
   - Admins review payment confirmations
   - Only approved contributions count

2. **Hidden Sensitive Information**
   - Phone numbers private (unless shared)
   - Transaction IDs visible to admins only
   - Payment details encrypted

3. **Audit Logs**
   - Complete record of all actions
   - Who approved/rejected what and when
   - Timestamps on every transaction
   - Transparent history for dispute resolution

4. **Contribution Limits**
   - Daily/monthly contribution limits
   - Suspicious activity flagged
   - Admin review for unusual amounts

5. **Group Verification**
   - Group creators provide verification details
   - Admin approval before group goes live
   - Member vetting process

## üÜò Troubleshooting

### Email Validation Error
**Problem:** "Invalid email" error on signup
**Solution:** 
- Use a valid email format: user@example.com
- Remove extra spaces before/after email
- Check email doesn't have special characters

### App Icon Not Showing
**Solution:** 
- Clear browser cache
- Hard refresh (Ctrl+F5 or Cmd+Shift+R)
- Restart the app

### Contribution Not Approved
**Steps:**
1. Check admin comments on submission
2. Provide correct transaction ID
3. Resubmit with clearer confirmation
4. Contact group admin for assistance

### Missing Mobile Money Message
**Alternative:**
- Use Manual Entry option
- Upload screenshot for verification
- Admin will review and approve
